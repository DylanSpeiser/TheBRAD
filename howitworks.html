How it Works:


Enter a Braille character on the keypad

Just copy the bumps using the switches!


Here's what's happening inside:

BRAD understands the status of all 6 switches as a 6-bit binary word.
It uses this value to index an array with 64 elements.
The standard Braille table is implemented by the placement of each letter in this array.


Press a button on the side

The raised button will finish the word and speak it, while the other button just adds that letter to the word.

If you want to just hear that one letter, just press the raised button.


BRAD's speech synthesizer handles the rest!

BRAD is equipped with a Parallax Emic 2 Text to Speech module, which handles converting the text input into human-understandable English speech.
